## Installation
### With Cargo
You can install [the ``tungsay`` crate](https://crates.io/crates/tungsay) from the official sources:
```sh
cargo install tungsay
```

### On Windows
Still in review

### On Arch btw
You can install [the ``tungsay`` package](https://aur.archlinux.org/packages/tungsay) from the official sources using an AUR Helper such as ``yay``:
```sh
sudo pacman -S yay
yay -S tungsay
```

### On Ubuntu/Debian
Kill yourself

## Usage
```sh
tungsay "Hello World!"
```
